The toolset

What the tools can reach.

MCP is only as capable as what it exposes. Bluestone's tools cover the full catalogue, read and write, so an agent can carry a whole process end to end, not just look things up.

Product & catalogue

read & write
Search products, then fetch full details by ID: names, attribute values, category assignments. Create and update products, attributes, variants, relations and catalogue structure. Agents write, not just read.

Media bank

assets
Upload assets and manage names, descriptions, labels and publish state programmatically, with no manual bank-by-bank work.

AI features

enrich · translate · analyse
Call AI Enrich to generate attribute values, AI Linguist to translate and rewrite (reading source and writing results back in one step), and AI Analyst to score products against golden-standard examples and suggest fixes.

Tasks & workflow

move work forward
Create, assign and complete tasks, add comments, and link affected products, so an agent turns findings into work the team can act on.

Quality & lineage

grounded actions
Read completeness scores, requirement breakdowns, validation issues and change history, so every action is grounded in real state and stays auditable.

Publishing & sync

get it live
Bulk-change status, publish across contexts, and sync to the delivery API: the last step from ready data to live listing.
Use cases

Eight workflows, run from a single prompt.

Each of these normally takes a product manager hours of manual, multi-step work. With MCP tools they become one conversation: the agent orchestrates the steps, you approve the outcome.

01

Completeness-driven publishing

Find near-ready drafts, publish those that qualify, and turn the rest into a task with a per-product list of what's missing.

"Find all draft products in the 'Winter 2026' category with a completeness score above 80% and publish those above 90%. For the rest, create a task for Sarah with details on what's missing."
02

Multi-language enrichment

Find products enriched in one language but empty in another, then translate, or generate from scratch where content is missing entirely.

"Find all published products that have an English product description but no German description, generate German translations, and update the products."
03

Campaign & promotion setup

Scope the products, create and apply a campaign label, set promotional attributes, and publish across the right contexts in one pass.

"Set up a Summer Sale campaign: find all outdoor and activewear products, label them 'Summer Sale 2026', set the promotional attribute to 'Up to 30% off', and publish them in the UK and DE contexts."
04

Intelligent variant grouping

Spot standalone products that belong together, propose the groupings for your confirmation, then build the variant groups.

"Find all single products added today in the 'Footwear' category and group them into variant groups based on their name and size/color attributes."
05

Cross-catalogue categorisation

Map newly imported products to the best-fit categories in a second catalogue, using attributes and the source category path.

"I've just imported 200 new products into the Master catalog. Assign them to the best-matching categories in our Amazon catalog based on their attributes and master category path."
06

Data quality audit

Combine structural gaps (completeness, validation) with semantic issues from AI Analyst, then raise a task per issue type.

"Audit all published products in the 'Electronics' catalog and give me a summary of the top data quality issues. Create tasks for each issue type and assign them to the appropriate team."
07

Automated relation linking

Match accessories to the products they fit by brand and compatibility, then create the relations in bulk once approved.

"Find all headphone accessories and automatically link them to compatible headphone products using the 'Compatible with' relation, based on brand and product family attributes."
08

Product lifecycle cleanup

Find stale or discontinued products, unpublish and sync any still live, then archive the set once you confirm.

"Archive all draft products that haven't been updated in over 6 months and are in the 'Discontinued' category. Make sure any that are still published are unpublished and synced first."
Safe by design

Powerful for agents. Controlled by you.

Giving an agent the run of your catalogue only works if it's governed. The same platform that exposes the tools also sets the guardrails.

Granular permissions

Agents act within scoped access, not one all-or-nothing admin key. They reach only what you allow.

Validation before it's live

Data is checked the moment it changes, so an agent can't publish incomplete or contradictory product data.

Full audit trail

Every action is logged with lineage and history, across people and agents alike, so you can trace and reverse anything.
