What agents can do

Operations your agents can run. Out of the box.

Once connected via MCP, agents can read and write across your data model programmatically — in automated pipelines, A2A workflows, or through any MCP-capable tool. No custom integration required for any of this.

Audit completeness across markets

Query completeness scores per context, surface attribute gaps by catalogue or supplier, and route the results to the next step in the pipeline.

Enrich and write back at scale

Read products with gaps, generate attribute values or descriptions, and write them back — across any context, without leaving the agent workflow.

Build and reorganise catalogue structure

Create category nodes, assign products, update hierarchy and membership — structural changes that would take hours in the UI, done in a single operation.

Validate compliance readiness

Check products against completeness templates — EU DPP, channel requirements, internal standards — and return structured gap reports for downstream action.

Compare data across contexts

Read and diff values across language or market contexts — completeness, attribute coverage, published state — without any custom reporting logic.

Trigger publishing and sync

Initiate channel syncs, retrieve publish history, inspect diffs — and gate the trigger on upstream conditions, so publication only happens when the data is ready.

For developers and power users looking to bring Bluestone PIM into their own AI workflows — Claude, Cursor, or ChatGPT — the frontier is at Bluestone PIM Labs.

From Labs to production

The path to Extensions Hub

Labs is the on-ramp. If what you build gets traction, the next step is the Bluestone PIM Extensions Hub — the marketplace where integrations and tools reach the full Bluestone PIM customer base, with the option to monetize.

Getting onto Extensions Hub requires passing Bluestone PIM's QA process, which checks for security, performance, and best practices. Labs makes that easier: community projects built here follow a lightweight version of the same standards, so there are fewer surprises when you go through formal review.
